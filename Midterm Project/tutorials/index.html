FTP Bruteforce Attack

This is an ftp brute-forcing tutorial I found on youtube created by Vari0us Hacking. I found it pretty helpful for network security beginners that want to learn a simple bruteforce attack. Vari0us Hacking uses nmap, one of the most commonly used security scanners, and an essential part of any hacker's arsenal. It is extremely versatile as it features a wide range of network security utilities in a lightweight application. It allows various functionalities, ranging from simple network scans, to custom scripts created for a specific purpose. In this tutorial, nmap is used to check if port 21 (FTP) is open on the victim machine. Then, nmap is used in conjunction with one of its default scripts, ftp-brute, to perform an exhaustive brute-force attack on the victim, trying random passwords until it finds a match.

This attack can usually be carried out on host machines or servers that do not have any kind of FTP security. However, most websites guard against these kinds of attacks by filtering their FTP port, limiting the number of FTP connections by the same IP address, and limiting the number of password attempts allowed. Therefore, although this attack would very likely fail on a real website, it's a great experiment for the network security beginner to perform and analyse, possibly on their own virtual machine.
